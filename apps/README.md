# RoofFlow AI - Roofing Business Dashboard

RoofFlow AI is a modern SaaS platform designed for roofing companies to manage leads, appointments, and gain AI-powered business insights.

## Features

- **Lead Management**: Track prospects from discovery to contract.
- **Appointment Scheduling**: Manage site visits and estimates with a clean calendar interface.
- **AI Business Insights**: Get actionable advice based on your business data (powered by GPT-4).
- **Authentication**: Secure sign-in with email and social providers.
- **Modern UI**: Dark mode, fully responsive, and high-performance.

## Tech Stack

- **Frontend**: React, Tailwind CSS, Lucide Icons
- **Backend**: Node.js (API Routes), PostgreSQL (Neon)
- **State Management**: React Query, Zustand
- **Auth**: Built-in Auth System (NextAuth based)
- **AI**: ChatGPT Integration

## Getting Started

1.  **Environment Variables**: Copy `.env.example` to `.env` and fill in your keys.
2.  **Database**: The schema is automatically handled by the Anything platform.
3.  **Authentication**: Enable Google/Email auth in the Anything project settings.
